# Чеклісти для тестування
## Модуль: Управління викладачами та кафедрами

### Функціональність
- [ ] Створення викладача з заповненням усіх полів.
- [ ] Перевірка вибору кафедри зі списку, що випадає (Dropdown).
- [ ] Робота пошуку за прізвищем у таблиці викладачів.
- [ ] Сортування таблиці за колонками (Прізвище, Ім'я, Кафедра).
- [ ] Натискання кнопки "Очистити" (скидання заповнених полів форми).
- [ ] Перевірка пагінації: зміна кількості рядків на сторінці (10, 20, 50).

### Негативні сценарії
- [ ] Спроба зберегти викладача без вказаного Прізвища (валідація червоним).
- [ ] Введення некоректного формату Email (наприклад, `user@@test..com`).
- [ ] Спроба створити кафедру з уже існуючою назвою (дублікат).
- [ ] Видалення кафедри, до якої прив'язані викладачі (перевірка попередження).
- [ ] Введення у текстові поля довгих рядків (більше 500 символів).

### UI/UX
- [ ] Перевірка відображення довгих назв кафедр у картках (чи не накладається текст).
- [ ] Візуальна підсвітка обов'язкових полів при помилці.
- [ ] Наявність іконок дії (око, олівець, смітник) та їх зрозумілість.
- [ ] Зміна кольору кнопок "Зберегти/Очистити" при наведенні (hover effect).
- [ ] Адаптивність: коректне відображення списку викладачів на екрані ноутбука.
